# matlab-mchv230vac1.5kw-33ak128mc106-acim-foc-vf v1.0.1
### Release Highlights
This is the second version of MATLAB-Simulink Model for ACIM Control: Sensored FOC (Incremental Encoder), Sensorless FOC, and V/F Control: MCHV-230VAC-1.5kW Development Board and dsPIC33AK128MC106 MC DIM. 
The model is set up for running a LEESON 3-Phase Induction Motor (230V, 60Hz, 1/3HP, 1.4A) mounted with a Phoenix America H9 Series Incremental Magnetic Encoder (H9-0512-0500-05-A-Y-A-F-A-X). The motor is tested under no load conditions. To achieve optimal performance under loaded conditions, the control parameters in the model may need additional tuning.
